CONFIGURATION:
================================================================================

Edit .rpi_config to change:
- RPI_USER (default: pi)
- RPI_HOST (your RPi IP/hostname)
- RPI_PROJECT_DIR (default: ~/plankton)

================================================================================
TROUBLESHOOTING:
================================================================================

Can't connect?
‚Üí Check RPi is on and connected to network
‚Üí Try IP instead of hostname: ssh pi@192.168.1.XXX
‚Üí Enable SSH: sudo raspi-config ‚Üí Interface ‚Üí SSH

Camera not working?
‚Üí Check cable connection
‚Üí Enable: sudo raspi-config ‚Üí Interface ‚Üí Legacy Camera
‚Üí Test: vcgencmd get_camera

Dependencies missing?
‚Üí SSH to RPi: ssh pi@raspberrypi.local
‚Üí Run: cd ~/plankton && ./setup_rpi.sh

Model file missing?
‚Üí Copy from laptop:
‚Üí scp "Downloaded models/best.pt" pi@raspberrypi.local:~/plankton/Downloaded\ models/
